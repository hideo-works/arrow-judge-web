## Arrow Judge

**Arrow Judge** is open source online judge system.

## Requirements
 * Apache with `mod_rewrite`
 * PHP 5.2.8 or higher
 * MySQL 5.0 or higher

## Installation
 * Extract the archive and upload to your server.
 * Visit your server with your browser and follow the instructions/
